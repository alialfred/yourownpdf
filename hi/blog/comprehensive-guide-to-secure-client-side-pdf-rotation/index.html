4. YourOwnPDF से PDF को Permanently Rotate कैसे करें? (स्टेप-बाय-स्टेप गाइड)

YourOwnPDF पर rotate pdf pages permanently करना बेहद आसान है। चूंकि यह पूरा प्रोसेस आपके ब्राउज़र में ही जावास्क्रिप्ट (JavaScript) की मदद से होता है, इसलिए यह सुपरफास्ट है। नीचे दिए गए चरणों का पालन करें: [ फ़ाइल चुनें / ड्रैग करें ] ➔ [ ओरिएंटेशन बदलें (90°/180°/270°) ] ➔ [ सुरक्षित डाउनलोड ]

स्टेप 1: अपनी फ़ाइल अपलोड करें

अपनी फ़ाइल अपलोड करेंYourOwnPDF.com के Rotate PDF tool पर आएं। अपने सिस्टम से PDF फ़ाइल को सीधे ड्रैग करके डैशबोर्ड पर ड्रॉप करें, या 'Select File' बटन पर क्लिक करके फाइल चुनें। (अधिकतम फ़ाइल साइज़ 100MB तक समर्थित है)।

YourOwnPDF का दृश्य इंटरफ़ेस जहाँ आप प्रत्येक पेज को रोटेट कर सकते हैं।

स्टेप 2: पेजों का ओरिएंटेशन चुनें

पेजों का ओरिएंटेशन चुनेंजैसे ही फ़ाइल लोड होगी (जो कि पलक झपकते ही हो जाती है क्योंकि कोई सर्वर ट्रांसफर नहीं होता), आपको सभी पेजों के थंबनेल दिखाई देंगे। आप प्रत्येक पेज के नीचे दिए गए रोटेशन आइकॉन पर क्लिक करके उसे rotate pdf clockwise या rotate pdf counter clockwise घुमा सकते हैं।

स्टेप 3: ड्रैग एंड ड्रॉप से रीअरेंज करें (वैकल्पिक)

ड्रैग एंड ड्रॉप से रीअरेंज करें (वैकल्पिक)अगर रोटेशन के साथ-साथ आपको पेजों का क्रम भी बदलना है, तो आप आसानी से किसी भी पेज को पकड़कर आगे-पीछे खिसका सकते हैं।

स्टेप 4: बदलाव सुरक्षित करें और डाउनलोड करें

बदलाव सुरक्षित करें और डाउनलोड करेंजब आप संतुष्ट हो जाएं कि सभी sideways scanned pdf और upside down pdf सीधे हो चुके हैं, तो 'Apply Changes' या 'Download' बटन पर क्लिक करें। आपका नया, पूरी तरह से ओरिएंटेड PDF तुरंत डाउनलोड हो जाएगा।
